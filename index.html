Intro

I'm a second-year Computer Science B.S. student at the University of California, Santa Cruz. I entered the tech world with a strong interest in mathematics and quickly became fascinated by how it drives fields like AI, machine learning, and algorithm design.

My journey began with Python, where I explored fundamental data structures and gained hands-on coding experience. As I progressed, I took courses like Analysis of Algorithms, which deepened my understanding of algorithmic strategies such as sorting, greedy methods, divide and conquer, dynamic programming, and recursion ‚Äî along with their time and space complexities.

One of my most rewarding experiences was collaborating on a team project to build an emulator for a fictional video game console, the Banana!, using C++. This project challenged us to work with binary parsing, real-time graphics rendering, and emulator design. As a team, we went beyond the base requirements by implementing extra features like a disassembler, pixel filter effects, game recording and playback, and a custom GUI.

Throughout other systems programming projects, one of the most valuable lessons I learned was the importance of writing strong and thorough test cases. For example, when building an HTTP server, a client-server system, and a memory buffer, I developed automated test cases using .sh scripts and structured file inputs to ensure robustness and correctness. Writing good tests became one of the most important habits I formed as an engineer.

Currently I'm passionate about exploring the intersection of software engineering and AI, and I'm always open to learning, collaborating, and diving deeper into research opportunities.
